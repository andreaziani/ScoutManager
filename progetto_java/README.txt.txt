Guida all'utilizzo dell'applicativo:
1) L'applicazione si collega alla base dati locale di MS SQL Server, per questo motivo per utilizzare il file jar è necessario:
- importare in MS SQL server la base dati di nome SCOUT.bacpac presente nella cartella.
- impostare la password di accesso del "sa" ad "1234".
- assicurarsi che il database locale permetta l'accesso con connessioni remote.

2) Utilizzo dell'applicazione vera e propria:
- Per accedere alla vista dell'admin utilizzare l'username: admin con password: admin.
- Per accedere alla vista del responsabile di evento nazionale utilizzare l'username e password di un reponsabile di evento nazionale registrato nella
  base dati ad esempio user: naz1 e password: naz1.
- Per accedere alla vista del responsabile di parrocchia utilizzare l'username e password di un responsabile di parrocchia registrato nella base dati
  ad esempio user: par1 e password: par1

3) Operazioni possibili:
- Tutte le operazioni possibili per ogni vista sono descritte nella relazione.
